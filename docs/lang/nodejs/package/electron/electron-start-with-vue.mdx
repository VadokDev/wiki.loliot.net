# Pre-installation

```shell
yarn global add @vue/cli
```

# Creat project

```shell
vue create <project>
```

```shell
cd <project>
```

```shell
yarn add electron --dev
```

```shell
vue add electron-builder
```

```shell
vue add vuetify
```

# Test running

```shell
yarn electron:serve
```

# Build

```shell
yarn electron:build
```

Then, check `dist_electron`.
